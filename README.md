# KOS Factory

KOS Factory is an extensible engineering platform for building, validating,
testing and evolving software systems.

Its first managed product is Knowledge Operating System (KOS), but the Factory
is designed to support any software project.

---

## Vision

Build once.
Reuse forever.

KOS Factory automates the software lifecycle:

- Project management
- Workspace management
- Code generation
- Validation
- Testing
- Documentation
- Release
- AI-assisted development
